About This Project

Women Safety Analytics – Protecting Women from Safety Threats is a college mini project developed using Flask, SQLite, HTML, CSS, JavaScript, and Bootstrap 5. The goal of this project is to create a simple and effective platform where incidents affecting women's safety can be reported, tracked, and analyzed.

The platform allows users to submit incident reports including the type of incident, location, date, and description. All reports are stored in a lightweight SQLite database. An analytics dashboard provides administrators or researchers with a quick overview of the number and types of incidents reported, using Chart.js visualizations such as pie charts and bar charts.

In addition, the platform includes an Emergency Contacts page listing important helpline numbers such as Police, Women Helpline, Ambulance, and Child Helpline, along with general safety tips to help raise awareness among users.

Technologies Used

  • HTML5, CSS3, JavaScript
  • Bootstrap 5 (CDN)
  • Python Flask (Backend Framework)
  • SQLite (Database)
  • Chart.js (Data Visualization)

Purpose

This project was built as an academic mini project to demonstrate full-stack web development skills while addressing a socially relevant issue — women's safety. It is intended for educational purposes and awareness building.
